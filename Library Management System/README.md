# Library Management System (Frontend)

A beginner-friendly Library Management System built using pure HTML, CSS, and JavaScript.  
This project allows users to add book details through a form and display them dynamically in a table without page refresh.

---

## Live Website
https://sujalpatil21.github.io/Web-Devlopment-Projects/Library%20Management%20System/

---

## Description

This project demonstrates the fundamentals of frontend web development by combining:

- HTML for structure  
- CSS for styling  
- JavaScript for interactivity  

Users can enter book information such as name, author, book ID, and category. When the Add Book button is clicked, the book is instantly added to the table.

---

## Files

- index.html — Main structure  
- style.css — Styling  
- script.js — Logic for adding books

- Add new book records

## How to Run Locally

1. Download or clone the repository  
2. Open the folder  
3. Double-click index.html  

---

## Example Usage

1. Enter book name  
2. Enter author  
3. Enter book ID  
4. Enter category  
5. Click Add Book  

Book appears in table.
